易感基因文件结果汇总

结果文件说明：
1.	readme.txt：文件详细介绍说明
2.	pre_gene.stat：易感基因统计表
3.	${tumor_sample}_${control}.merge_anno.txt/xls：样品易感基因注释文件

表头具体说明：
pre_gene.stat：易感基因统计表
1.	Sample_Control：肿瘤样品_对照样品；
2.	Gene_numbers：dbNSFP和CGC注释到的易感基因总数量；

${tumor_sample}_${control}.merge_anno.txt/xls：样品易感基因注释文件
1.	Chromosome:染色体；
2.	Pos:位置；
3.	Ref:参考基因组碱基类型；
4.	Alt:变异碱基类型；
5.	Rs_id:单核苷酸多态性（SNP）位点的标识符；
6.	MetaSVM_pred:用于预测某个变异位点是否对蛋白质功能产生影响。通常为T或D。T表示该变异位点预测为耐受性变异，即不会对蛋白质功能产生显著影响；D表示该变异位点预测为有害变异，即可能对蛋白质功能产生显著影响；
7.	FATHMM_pred: 用于预测某个变异位点是否对蛋白质功能产生影响。通常为T或D。T表示该变异位点预测为耐受性变异，即不会对蛋白质功能产显著影响；D表示该变异位点预测为有害变异，即可能对蛋白质功能产生显著影响；
8.	LRT_pred:用于预测某个变异位点是否对蛋白质功能产生影响。通常为D、N或U。D表示该变异位点预测为有害变异，即可能对蛋白质功能产生显著影响；N表示该变异位点预测为中性变异，即不会对蛋白质功能产生显著影响；U表示该变异位点预测结果不确定；
9.	PROVEAN_pred:用于预测某个变异位点是否对蛋白质功能产生影响。通常为T或D。T表示该变异位点预测为耐受性变异，即不会对蛋白质功能产生显著影响；D表示该变异位点预测为有害变异，即可能对蛋白质功能产生显著影响；
10.	MutationTaster_pred:用于预测某个变异位点是否对蛋白质功能产生影响。通常为A、D、N、P或者U。A表示该变异位点预测为致病性变异，即可能对蛋白质功能产生显著影响；D表示该变异位点预测为有害变异，即可能对蛋白质功能产生显著影响；N表示该变异位点预测为中性变异，即不会对蛋白质功能产生显著影响；P表示该变异位点预测为可能是有害变异；U表示该变异位点预测结果不确定；
11.	MutationAssessor_pred:用于预测某个变异位点是否对蛋白质功能产生影响。通常为H、M、L或者N。H表示该变异位点预测为高影响变异，即可能对蛋白质功能产生显著影响；M表示该变异位点预测为中等影响变异，即可能对蛋白质功能产生一定影响；L表示该变异位点预测为低影响变异，即可能对蛋白质功能产生较小影响；N表示该变异位点预测结果不确定；
12.	Polyphen2_HDIV_pred:用于预测某个变异位点是否对蛋白质功能产生影响。通常为B、P或者U。B表示该变异位点预测为可能是有害变异，且该变异位点在高度保守的区域中，即可能对蛋白质功能产生显著影响；P表示该变异位点预测为可能是有害变异，但该变异位点不在高度保守的区域中，即可能对蛋白质功能产生一定影响；U表示该变异位点预测结果不确定；
13.	Polyphen2_HVAR_pred:预测某个变异位点是否对蛋白质功能产生影响。Polyphen2_HVAR_pred是PolyPhen-2工具预测结果的一部分，通常为B、P或D。B表示该变异位点预测为边缘变异，即可能对蛋白质功能产生轻微影响；P表示该变异位点预测为可能有害变异，即可能对蛋白质功能产生显著影响；D表示该变异位点预测为有害变异，即可能对蛋白质功能产生非常显著影响；
14.	SIFT_pred:用于预测某个变异位点是否对蛋白质功能产生影响。SIFT_pred是SIFT工具预测结果的一部分，通常为T或D。T表示该变异位点预测为耐受性变异，即不会对蛋白质功能产生显著影响；D表示该变异位点预测为有害变异，即可能对蛋白质功能产生显著影响；
15.	Deleterious_count:在某个基因中被预测为有害的变异数量；	
16.	Cosmic_id:Cosmic数据库注释编号；
17.	Protein_change:蛋白质序列中的氨基酸变化；	
18.	AA_change:蛋白质序列中的氨基酸改变；
19.	AF_eas:在东亚人群中的等位基因频率；
20.	Effect:功效；
21.	Impact:影响；
22.	Gene:基因名；
23.	Geneid:基因id；
24.	CGC_Cancers:CGC数据库注释结果;
